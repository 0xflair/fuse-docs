1. To set up Fuse as a new network on Metamask. Click on the network selector at the top of the app and then choose "Custom RPC" from the list:\
   ![](/img/assets_-MRxFOj-hfekokMwcTbggN_-MVohzwcoaqm5IBcGGhf_-MVoj0Lx1BPLqn3p7MB__image.webp)
2. Then in the "New RPC URL" enter this address: New rpc URL: https://rpc.fuse.io

<Admonition type="info" title="Optionally you can add the full parameters:">
  Additional: [https://rpc.fuse.io\
  ](https://rpc.fuse.io)ChainId: 122
  Explorer: [https://explorer.fuse.io\
  ](https://explorer.fuse.io/)Symbol: Fuse
</Admonition>
