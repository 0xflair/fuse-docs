## Request Verify SMS



This is the subsequent call to the "Request SMS Verification Code" to verify the SMS code received to the provided phone number.

Here you also need to provide the user's account address, which will be included in the JWT that is returned when verification is successful.

Most subsequent calls in the Wallet API will require the JWT you receive back from this call.
